* 调式音阶(mode)
    * **`Ionian`**(伊奥尼亚调式——自然大调音阶): 1, 2, 3, 4, 5, 6, 7.
    * `Dorian`(多利亚调式——自然大调2级音阶): 1, 2, b3, 4, 5, 6, b7.
    * `Phrygian`(弗利几亚调式——自然大调3级音阶): 1, b2, b3, 4, 5, b6, b7.
    * `Lydian`(利底亚调式——自然大调4级音阶): 1, 2, 3, #4, 5, 6, 7.
    * `Mixolydian`(混合利底亚调式——自然大调5级音阶): 1, 2, 3, 4, 5, 6, b7.
    * **`Aeolian`**(爱奥里亚调式———自然小调式): 1, 2, b3, 4, 5, b6, b7.
    * `Locrian`(洛克利亚调式——自然大调7级音阶): 1, b2, b3, 4, b5, b6, b7.
    * 以上称中古调式/教会调式. 
* 音阶一定是调式, 调式不一定是音阶. 
* 同主音关系调

* 平行大小调(关系大小调)

    * 大调转小调: 3, 6, 7 (mi, la, si) 各降半音.

* 完全和谐音程: 
    * 纯1度: 齐奏. (如C-C)
    * 纯8度: 作用是加强气势. 
    * 纯4度: (如C-F)
    * 纯5度: 相差7个半音. (如C-G)
* 不完全和谐音程: 
    * 大3度: 相差4个半音. 
    * 小3度: 相差3个半音. 
    * 大6度: 相差9个半音. 
    * 小6度: 相差8个半音. 
* 不和谐音程: 大小2度和大小7度, 音程都是一个半音. 
* 三全音: 增4度/减5度, 有3个全音/6个半音的距离, 如C-F#. 
* 五度圈
    * 五度音: 三个全音加一个半音
* 和弦
    * 参考
        * [七和弦](https://zh.wikipedia.org/wiki/%E4%B8%83%E5%92%8C%E5%BC%A6)
    * 按三度关系叠置的和弦: 
        * 三和弦(triad)
            * 大(major): 大三度 + 小三度
            * 小(minor): 小三度 + 大三度
            * 增(augment): 大三度 + 大三度
            * 减(diminished): 小三度 + 小三度
        * 七和弦(seventh chord)
            * 较正常: 
                * 属七和弦(dominant ~)
                    * 别称: 大小七和弦.   
                    * 性质: 
                        * 大三和弦 + 小三度. 
                        * 根音与七音相距为小七度. 
                        * 将根音建立在属音上. 
                    * 例: Cdom7(`C, E, G, Bb`)(也叫C7)
                    * 效果: 具有不和谐性, 可以强调属音, 气氛略微紧张, 
                * 大七和弦
                    * 别称: 大大七和弦. 
                    * 性质: 
                        * 大三和弦 + 大三度.
                    * 例: Cmaj7(`C, E, G, B`)
                    * 没有多少倾向, 多数时候可以直接代替大三和弦. 
                * 小七和弦
                    * 别称: 小小七和弦. 
                    * 性质: 
                        * 小三和弦 + 小三度. 
                        * 根音与七音相距为小七度. 
                    * 例: Cm7(`C, Eb, G, Bb`)
                    * 多数时候可以直接代替小三和弦
                * 半减七和弦(half-diminished ~)
                    * 别称: 减小七和弦, 小七减五和弦, 导七和弦
                    * 性质: 
                        * 减三和弦 + 大三度. 
                        * 根音与七音相距为小七度. 
                    * 例: Cm7b5(或C half-dim(7), Cm7-5)(`C, Eb, Gb, Bb`)
                * 减七和弦(diminished ~) 
                    * 别称: 减减七和弦, 减导七和弦. 
                    * 性质: 
                        * 减三和弦 + 小三度
                    * 例: Cdim7(也叫C°7)(`C, Eb, Gb, A`)

            * 较特殊: 
                * 小大七和弦: 
                    * 性质: 
                        * 小三和弦 + 大三度.
                    * 例: CmM7(也叫cM7)(`C, Eb, G, B`)
                    * 效果: 小大七和弦的不协和性极强, 并且包含"大三度+大三度"的增音程结构, 故在古典主义时期的音乐中应用极少. 但在低沉阴郁的曲调中, 该和弦的七音(例如a和声小调中的升G)却可以充当和弦外音来保护旋律的完整性. 
                * 增小七和弦: 
                    * 性质: 
                        * 增三和弦 + 减三度. 
                        * 根音与七音相距为小七度. 
                    * 例: Caug7(也叫C+7)(`C, E, G#, Bb`)
                * 增大七和弦: 
                    * 性质: 
                        * 增三和弦 + 小三度
                    * 例: CM7+5(也叫Caug(maj7), C+(maj7), C+Δ7)(`C, E, G#, B`)
            * 实际演奏中, 大七, 小七, 属七的五音可以忽略. 
        * 九和弦
        * 十一和弦
        * 十三和弦
    * 由以上和弦发展来的: 
        * 六和弦
        * alt和弦
        * 各种变化音和弦
    * 不按三度关系叠置的和弦: 
        * 挂留和弦(suspended chord): 用二度音或者纯四度音代替原来的三度音而排列组合成的和弦. 
            * 挂二和弦, 它用二度音代替了原来的三度音, 从而形成了一种带有怀旧情绪的新和弦. 
            * 注: 挂几就是在和弦中加入一个几级的和弦外音.

## 声卡
 * 频率响应(Frequency Response)​​
    * ​​定义​: 声卡能处理的音频频率范围(单位: Hz), 以及在该范围内信号幅度的平坦度(以±dB表示波动). 
    * ​​标准范围​: 理想值为 ​​20Hz–20kHz​​(覆盖人耳听觉极限), 高端声卡可达 ​​5Hz–50kHz​​. ​​实际影响​: 
    * ​​波动值​: 如标注"20Hz–20kHz ±0.5dB", 表示全频段幅度偏差≤0.5dB, 频响曲线越平直, 音色越真实. 
    * ​​低频延伸​: 低于20Hz影响声场深度(如鼓声余震)；​​高频延伸​: 高于20kHz提升泛音空气感(如镲片细节). 
    * ​​选购参考​: 专业声卡需标注波动值(±dB), 未标注波动值的数据无意义. 
* 噪声水平(Noise Level, dBA)​​
    * ​​定义​: 声卡自身产生的底噪强度, 以A计权分贝(dBA)测量(A计权模拟人耳对噪音的敏感度). 
    * ​​专业级指标​: ≤ ​​-120dBA​​(如RME ADI-2 Pro为-120dBA), 入门级约-100dBA. 
    * ​​影响因素​: 
    * ​​电路设计​: 分立元件/屏蔽设计可降低热噪声. 
    * ​​增益设置​: 高增益下噪声更显著(如话筒输入放大60dB时). 
    * ​​听感表现​: ＞-100dBA时, 安静段落可能听到"嘶嘶"声(尤其耳机监听). 
* 动态范围(Dynamic Range, dBA)​​
    * ​​定义​: 最大不失真信号与底噪的比值(单位: dBA), 反映声卡同时呈现极弱与极强信号的能力. 
    * ​​专业级指标​: ≥ ​​115dBA​​(如Apogee Symphony I/O达129dBA), CD标准为96dBA. 
    * ​​实际意义​: 
    * ​​高动态范围​: 保留音乐中的细微呼吸声与爆发性鼓点(如交响乐高潮). 
    * ​​低动态范围​: 强信号被压缩(失真), 弱信号被噪声淹没. 
* 总谐波失真(THD, %)​​
    * ​​定义​: 输入纯正弦波时, 声卡输出中产生的额外谐波成分占总信号的比例. 
    * ​​专业级指标​: ≤ ​​0.001%​​(如Metric Halo ULN-8为0.0008%). 
    * ​​谐波类型​: 
        * ​​奇次谐波​​(3次, 5次): 刺耳感(如吉他过载). 
        * ​​偶次谐波​​(2次, 4次): 温暖感(部分电子管设备刻意保留). 
* THD+N(THD + Noise, dBA)​​
    * ​​定义​: 总谐波失真+底噪的综合指标, 比单独THD更能反映实际听感问题. 
    * ​​专业级指标​: ≤ ​​-100dBA​​(相当于0.001%失真+底噪). 
    * ​​测试方法​: 输入1kHz正弦波, 测量输出信号中的失真+噪声总能量. 
    * ​​重要性​: 直接影响人声/乐器的"纯净度", 劣质声卡易产生"毛刺感". 
* 互调失真+噪声(IMD + Noise, %)​​
    * ​​定义​: 输入多频混合信号时, 因电路非线性产生的​​新频率成分​​(非谐波)占总信号的比例. 
    * ​​专业级指标​: ≤ ​​0.005%​​(如Antelope Audio Orion 32+为0.0007%). 
    * ​​听感影响​: 导致声音"浑浊", 尤其在高频复杂段落(如钢琴和弦). 
* 立体声串扰(Stereo Crosstalk, dBA)​​
    * ​​定义​: 左声道信号泄露到右声道的强度(反之亦然), 以分贝值表示隔离度. 
    * ​​专业级指标​: 1kHz时≤ ​​-100dB​​(如RME Babyface Pro FS为-110dB). 
    * ​​问题表现​: ＞-60dB时声像定位模糊(如鼓组位置不清晰). 
* 10kHz互调失真(IMD at 10kHz, %)​​
    * ​​定义​: 针对高频信号(如10kHz)的互调失真测试, 反映声卡对高频的保真能力. 
    * ​​专业级指标​: ≤ ​​0.003%​​(高端声卡如Prism Sound Titan可达0.0005%). 
    * ​​高频重要性​: 10kHz是镲片, 小提琴泛音的核心频段, 失真会削弱"光泽感".

## MIDI
* MMA: MIDI生产商联合会
* AMEI: 日本音乐电气工业协会
* GM1(通用MIDI 1.0规范化标准)
    * 1995确立. 为了在最低的容量及其他硬件条件下获得最好的乐器音色表现. 
    * 一些规范: 
        * 标准部件
            * MIDI线缆
            * 标有IN, OUT, THRU的MIDI端口
        * 每个MIDI端口16个MIDI通道, 每个通道负责一个声部(对应一个音色). 
            * MIDI通道10为打击乐器专用通道. 
        * 每个音色库最大包含128个音色, 每个音色有一个固定名称和ID(0~127). 
        * 控制码组有128个控制码. 
* GM2
    * MMA和AMEI共同主导; 1999
    * 增加更多音色和控制
    * 对音色发声单元部分构建了一个规范化的框架
    * 兼容GM1
* DLS(Downloadable Sounds, 可载入音色的MIDI)
    * 使MIDI数据能够携带一段音乐所需要的音色样本
* XG(扩展GM标准)
    * Yamaha制定
* GS(通用标准MIDI)
    * Roland开发制定
    * 采用标准化方案, 并定义了新的MIDI表现控制功能, 如音色的用户编辑, 效果器, 更丰富的音色组合. 
    * 音色编号为1~128
